# 更新歷程 (Changelog)

這份文件記錄了 Floating-Lyrics 專案所有的更新紀錄與新功能。

## 最新更新：Web 版進階功能與排版修復

### 🌟 新功能 (New Features)
- **Web 版支援單擊修改假名**：現在在網頁版的歌詞面板中，只要「直接點擊漢字」，就會自動彈出假名編輯視窗，不需要再點擊進階選項按鈕了！
- **羅馬拼音自動轉換**：在網頁版的假名編輯視窗中，輸入羅馬拼音（Romaji）時，系統會「即時自動轉換」為平假名（Hiragana），大幅加快修改速度！

### 🐛 錯誤修復 (Bug Fixes)
- **修復假名修改導致排版崩壞的問題**：
  - **原因**：之前儲存修改時，會不小心把 HTML 標籤（用來顯示假名的排版代碼）存入資料庫的 `lyrics` 資料表中。導致下一次注音轉換程式 (`furigana_inject.py`) 讀取時，把 HTML 標籤當作日文字處理，造成標籤無限嵌套、排版崩潰。
  - **解法**：修改儲存邏輯，現在只會呼叫 `/api/furigana/correct` 將使用者的修改正確存入 `word_corrections`（單字修正庫）資料表，再也不會污染原始歌詞資料。
- **修復進階選項視窗被遮擋的問題**：補齊了 `index.ejs` 中遺漏的 `</div>` 標籤，確保進階選項視窗不會被其他元件擋住或隱藏。
- **優化假名排版行高**：加大了網頁版歌詞每行的 `line-height`，確保假名（`rt`）有足夠的垂直空間，不會再與上一行疊在一起。

### 📊 其他改動
- 實作了完整的網頁版統計面板（Stats & Leaderboard），可顯示聽歌總時長、熱門歌手與熱門歌曲排行。
